# Data

`scripts/run_analysis.py` downloads the original SECOM files into `data/raw/` when they are missing.

Original source:

- https://archive.ics.uci.edu/dataset/179/secom
- https://archive.ics.uci.edu/ml/machine-learning-databases/secom/secom.data
- https://archive.ics.uci.edu/ml/machine-learning-databases/secom/secom_labels.data

The raw data is a public semiconductor manufacturing process sensor dataset. Feature names are anonymized, so reports must not claim confirmed physical root causes.
